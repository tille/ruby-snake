Ruby Snake
==========

Snake is an open source 2D game that was created with Ruby using [Gosu](http://www.libgosu.org/)

## Setup
To install all the dependencies, run:

```
bundle install
```

Next using foreman we can boot the application.

```
$ foreman start
```
